Змейка (Игра на C++)
Описание
Классическая игра "Змейка" (Snake), написанная на C++ для Windows. Игрок управляет змейкой, собирает еду и старается не врезаться в стены или в собственный хвост.

Особенности
Простое управление (WASD или стрелки)

Система очков и длины змейки

3 уровня сложности

Персонализация имени игрока

Консольная графика

Автосохранение рекордов

Установка и запуск
Требования:
Windows 7/8/10/11

Visual Studio 2017 или новее

Или любой компилятор C++ с поддержкой Windows API

Сборка через Visual Studio:
Скачайте исходный код

Откройте проект в Visual Studio

Нажмите F5 для компиляции и запуска

Сборка через командную строку:
bash
g++ main.cpp -o SnakeGame.exe -std=c++11
Управление
Клавиша	Действие
W или Стрелка вверх	Движение вверх
S или Стрелка вниз	Движение вниз
A или Стрелка влево	Движение влево
D или Стрелка вправо	Движение вправо
Q или ESC	Выход из игры
P	Пауза
Правила игры
Управляйте змейкой с помощью клавиш

Собирайте еду (символ F) для увеличения счета

Каждая собранная еда:

Увеличивает счет на 10 очков

Увеличивает длину змейки на 1 сегмент

Избегайте:

Столкновения со стенами

Столкновения с собственным хвостом

Игра заканчивается при столкновении

Уровни сложности
Уровень	Скорость	Описание
Легкий	Медленно	Подходит для начинающих
Средний	Нормально	Стандартная скорость
Сложный	Быстро	Для опытных игроков
Система очков
Действие	Очки
Собрать еду	+10
Длина > 10	×1.5 множитель
Длина > 20	×2.0 множитель
